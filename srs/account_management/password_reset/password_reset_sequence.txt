# Passwort Reset Sequence Diagram

title Passwort zurücksetzen

participant Spieler
participant Client
participant Server
participant Datenbank

activate Spieler
Spieler->Client:"Passwort vergessen" geklickt
activate Client
Spieler->Client:E-Mail-Adresse eingegeben
Spieler->Client:Schaltfläche "Passwort zurücksetzen" gedrückt
Client->Server:Passwort-Reset \nE-Mail anfordern
activate Server
Server->Datenbank:E-Mail prüfen
activate Datenbank

alt E-Mail gefunden
  Datenbank-->Server:E-Mail bestätigt
  deactivate Datenbank
  note over Server:E-Mail gesendet
  Server-->Spieler:Bestätigung anzeigen
  deactivate Server
  note over Spieler:Neues Passwort in E-Mail
  
else E-Mail nicht gefunden
  Datenbank-->Server:E-Mail nicht gefunden
  deactivate Datenbank
  Server-->Spieler:Bestätigung anzeigen \n(aus Sicherheitsgründen)
  deactivate Server
end

deactivate Client
deactivate Spieler
