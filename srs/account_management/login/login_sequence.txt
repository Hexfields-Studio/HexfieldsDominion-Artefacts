participant Spieler
participant Client
participant Server
participant Datenbank

activate Spieler
Spieler->Client:Anmeldedaten eingegeben
activate Client
Spieler->Client:Schaltfläche "Log in" gedrückt
Client->Server:Anmeldedaten \nwerden übermittelt
activate Server
Server->Datenbank:Anmeldedaten \nwerden übermittelt
activate Datenbank

note over Datenbank:Anmeldung prüfen

Datenbank-->Server:Anmeldung bestätigen
deactivate Datenbank
Server-->Client:Anmeldung bestätigen
deactivate Server

note over Client:Anmeldedaten \nwerden gespeichert

deactivate Client
deactivate Spieler
