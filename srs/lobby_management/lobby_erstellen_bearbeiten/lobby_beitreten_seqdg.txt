title Lobby beitreten

participant Spieler
participant Frontend
participant Backend

activate Spieler
Spieler->Frontend: Klickt "Beitreten"
activate Frontend
Frontend->Backend: Zum Code zugehörige Lobby abfragen
activate Backend
Backend->Frontend: Zugehörige Lobby mit ID zurückgeben
deactivate Backend
Frontend->Spieler: Lobby Seite anzeigen
deactivate Frontend
deactivate Spieler
